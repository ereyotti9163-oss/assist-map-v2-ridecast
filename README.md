​主な機能
<img width="300" height="1560" alt="image" src="https://github.com/user-attachments/assets/257537b3-e11c-478d-93a8-a48327c3af7a" />

​リアルタイム風速・風向表示: OpenWeatherMap API（または使用中のAPI名）と連携し、現在地の詳細な風情報を取得。
​サイクリングロード特化マップ: サイクリストが走りやすいルートを優先的に確認可能。
​PWA対応: スマートフォンのホーム画面に追加することで、ネイティブアプリのように素早く起動できます。
​Heading Up機能: 進行方向（デバイスの向き）に合わせてマップが回転し、直感的なナビゲーションを実現。
​使い方
​アプリを開き、位置情報の使用を許可します。
​画面右下の [風向] ボタン をタップします。
​画面上部に現在の「風速」と「風向き」のアイコンが表示されます。
​技術スタック
​Frontend: HTML5, CSS3, JavaScript (Vanilla JS)
​Maps: Leaflet.js / OpenStreetMap
​Data: OpenWeatherMap API (Current Weather Data)
​Deployment: GitHub Pages
​インストール方法
​このアプリはPWAです。以下の手順でスマホのホーム画面に追加して使用してください。
​iOS (Safari): 共有ボタン ＞ 「ホーム画面に追加」
​Android (Chrome): メニューボタン ＞ 「アプリをインストール」または「ホーム画面に追加」
